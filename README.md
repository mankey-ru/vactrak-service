# vactrak-service

Vacancy tracker service for personal needs.

Works together with [userscript for HeadHunter](https://github.com/mankey-ru/userscripts/blob/main/dist/hh-vactrak.user.js).

After new vacancy is published, the app will immediately send TG message. For getting TG notifications, you should open https://t.me/vactrak_bot and press Start.
